Parte 2: Questões Discursivas

6 Defina e diferencie os tipos de dados estruturados, semi-estruturados e não estruturados, destacando como cada categoria se organiza e é armazenada. Cite pelo menos um exemplo real de formato ou tecnologia associada a cada tipo. (3 pontos)
Estruturados => Usado para conjuntos de dados com pouca ou nenhuma variação. Armazenado em tabelas. Ex: Excel. Semi-Estruturados => Usado para conjuntos de dados com um considerável volume de variação. Armazenagem mais complexa. Ex: JSON, XML, MongoDB. Não-Estruturados => Conjunto de dados com múltiplas variações e dinâmicas. Ex: áudios, vídeos.
7 Explique o que são métricas e indicadores-chave de desempenho (KPIs), destacando a diferença entre ambos e apresentando exemplos práticos de cada um em um contexto organizacional ou de projeto. (3 pontos)
As Métricas são os dados brutos retirados de uma fonte de dados do projeto. Através das métricas é possível o levantamento de indicadores, entre eles os KPIS que visam analisar o desempenho do projeto frente às metas e objetivos. Exemplo de métrica: Total de vendas ou faturamento da campanha ou da taxa de conversão de clientes. Exemplos de KPI: Aumento / Diferença da taxa de conversão mensal, usada para medir o desempenho frente a meta de aumentar em 25% a taxa de conversão de novos clientes.
8 Descreva o processo de levantamento de métricas SMART, explicando o que representa cada elemento do acrônimo (Specific, Measurable, Achievable, Relevant, Time-bound) e como esses critérios auxiliam na criação de métricas e KPIs mais claros e mensuráveis dentro de um projeto. (3 pontos)
As métricas são avaliadas para saber se se enquadram nos parâmetros para uma boa métrica. São esses parâmetros: S - Ter sentido e ser única / palpável. M - Ser capaz de avaliar progressos e variações. A - Possível de concluir. R - Deve haver e fazer parte do contexto. T - Deve ter um tempo limite. Seguindo esses parâmetros é possível levantar ótimas métricas e indicadores com visão para metas e objetivos.

10 Os processos de análise de dados envolvem diversas etapas, e uma das mais importantes é a integração e centralização das informações provenientes de diferentes sistemas, setores e períodos. Essa centralização permite que os sistemas de apoio à decisão utilizem dados unificados para gerar análises mais consistentes. Explique qual tecnologia é responsável por centralizar os dados de vários sistemas de uma organização e descreva sua importância para o processo de análise e tomada de decisão. (3 pontos)
A Data Warehouse é o "sistema" que guarda as informações de forma mais centralizada e tratada, onde os dados chegam através de um outro sistema, o ETL. O DW é importante pois facilita a análise e montagem de métricas e indicadores para o levantamento de metas e objetivos. Sem essa tecnologia, seria necessária a análise de dados brutos e altamente complexos, o que traria um maior custo para erros, apurações, afinal, disco é mais barato que processamento.
